# Star Notary Dapp

## Install

```
cd star_notary
npm install 

# run tests
truffle devleop
> test  # in truffle console, exit to proceed (ctrl+c) 

# if want to deploy new contract
truffle compile
truffle migrate --network sepolia 

# fontend is running on lite-server 
npm install dev
```
